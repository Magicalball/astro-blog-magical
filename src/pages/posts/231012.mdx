**解题思路**

思路

代码一：

```
#include <bits/stdc++.h>
using namespace std;
int main()
{
    int k, n;
    double sum = 0;
    cin >> k;
    for (n = 1;; n++)
    {
        sum +=  1.0 / n;
        if (sum > k)
        {
            break;
        }
    }
    cout << n;
}
```
