# 2、安装mysql

在窗口中运行如下指令：

```
docker pull mysql:latest

```

<Image src="/images/posts/221013/7a44fd4feab7434d887130d0d3e00237.png" width="700px" />

然后输入这一行 其中123456代表mysql初始密码

```
docker run -itd --name mysql-test -p 3306:3306 -e MYSQL_ROOT_PASSWORD=123456 mysql
```

<Image src="/images/posts/221013/bd0021dd8fad49b988a5b64ec68e5105.png" width="700px" />

使用以下命令配置访问权限

###### 进入容器

docker exec -it mysql-test bash

###### 登录mysql

mysql -u root -p  
123456

###### 设置mysql远程权限

ALTER USER 'root'@'%' IDENTIFIED WITH mysql\_native\_password BY '123456';

###### 刷新权限

flush privileges;

  

<Image src="/images/posts/221013/549b222ae99c47caa4ca2f8c03ca0dfb.png" width="700px" />

 **至此mysql 安装结束**
